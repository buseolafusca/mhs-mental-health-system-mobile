# mhs-mental-health-system-mobile

Steps to run this project:
```
1. run: npm install
2. run: npm run build && npm run start-sw
3. go to the url shown in the logs
```

Commands to start up application locally:

```
npm install
npm install axios
npm run build
npm install -g serve
serve -s build
```
